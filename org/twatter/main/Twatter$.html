org.twatter.main

Twatter

object Twatter extends TwatterMainTrait

The main launcher script for the service. This parses the following command line options:

- h | help : prints this help text - v | version : prints the version of the server - r | redis : specify the redis uri to use - o | output : specify the output directory for files - p | poison : specify a poison word list - t | threads : specify the number of processing threads
